# Styling-RecipeWebpage
# About the Project
It is the continuation of my first project which is plain html Recipe webpage and now we design it using Css
# Features
- External CSS stylesheet
- Styled recipe header
- Ingredients section
- Instructions section
- Tips section
- Responsive layout for smaller screens
# Filed Used
-- `index.html` - contains the structure and content of the recipe page
- `style.css` - contains the styling for the webpage
- `ASSETS/` - contains the recipe image

## What I Learned
In this project, I learned how to:
- Link an external CSS file to an HTML file
- Use class and ID selectors
- Style text, colors, spacing, and layout
- Use CSS Grid and Flexbox
- Add hover effects and responsive design
